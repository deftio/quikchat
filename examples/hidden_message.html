QuikChat Message Visibility Demo

This demo shows how to use the message visibility feature. Messages can be added to the history but remain hidden from view by setting visible: false. This is useful for including contextual information, like a system prompt for an LLM, without cluttering the UI.

This chat has an initial hidden message (ID: 0) containing a system prompt. Use the controls to make it visible or to add new hidden messages.

Status messages will appear here.
