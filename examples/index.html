Examples

Working examples showing how to use QuikChat. Includes basic usage, theming, multiple chats, and LLM integrations.

Basic Usage

  • UMD (script tag) — theme switching, toggle controls
  • ESM (module import) — ES module usage
  • Dual Chatrooms — two widgets that send messages to each other

LLM Integrations

  • Ollama (simple) — local LLM chat with Ollama
  • Ollama with Memory — conversational context via historyGet()
  • LM Studio with Memory — local LLM via LM Studio
  • OpenAI — GPT-4o with token streaming

Markdown

  • Markdown (basic) — quikchat-md build: bold, italic, code blocks, tables, lists (~9 KB gzip, no network needed)
  • Markdown (full) — quikchat-md-full build: adds syntax highlighting, math, maps, SVG (~14 KB gzip, loads extras from CDN on demand)

Framework Integration

  • React — using quikchat inside a React component
